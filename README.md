# Claude Code Landing

Framer Motion landing page for Claude Code. Next.js 14 + Tailwind + Framer Motion.

## Run

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Stack

- Next.js 14 (App Router)
- Framer Motion 11
- Tailwind CSS 3.4
- Lucide icons
- TypeScript

## Sections

- Navbar — sticky, blurs on scroll
- Hero — animated gradient mesh, typewriter-style stat counters, install command
- Features — bento grid, scroll-triggered reveals
- Terminal Demo — live typing animation looping through agent script
- Pricing — 3-tier with featured glow
- CTA — animated gradient banner
- Footer
